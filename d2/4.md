#### 火力狗广播

![](../.gitbook/assets/QQ图片20211001123901.png)

{% hint style="info" %}
幽灵有摇修，把**需要护盾**打钩
{% endhint %}

指挥官一般会说：集火X开头。紧接着会说：加锁X开头。指挥官说的时候会广播出来

![](../.gitbook/assets/QQ图片20211001124224.png)

{% hint style="info" %}
右键或者按住锁定键并点击这条广播就可以锁定该目标
{% endhint %}

![](../.gitbook/assets/QQ图片20211001124708.png)
